HTML and CSS Reference Sheet

HTML Semantic Elements

Tag SemanticElement Description
<header> HEADER Container for introduction or set of navigation links
<main> MAIN Specifies the main content of a document
<section> SECTION Defines a section in a document
<nav> NAV Defines a set of navigation links
<figure> FIGURE Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
<figcaption> FIGCAPTION Defines a caption for a figure element
<aside> ASIDE Defines some content aside from the content it is placed in (sidebar)
<article> ARTICLE Specifies independent,self-contained content

CSS TRANSITIONS

property Values Description
transition-timing-function transition-timing-function: ease specifies a transition effect with a slow start, then fast, then end slowly (this is default)
transition-timing-function transition-timing-function: linear specifies a transition effect with the same speed from start to end
transition-timing-function ease-in-out Specifies a transition effect with a slow start and end
transition-delay transition-delay: 1s Specifies a delay (in seconds) for the transition effect
transition-property transition-property: width; Defines a property for transition
transition-duration transition-duration: 2s; Specifies the tranisition duration
transition transition: width 2s; transition property specifies both property and duration in one property
